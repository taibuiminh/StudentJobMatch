# STUDENT MATCH

## Motivation 
In this program, I was trying to analyze data from excel tables. 
The program allows you to get a list of the best jobs for a student.

## Build status 
Finished.
But I am waiting for more information and data sets from the B&B education company to improve my code.

## Code
Java 

## Used Tech 
Poi library for analysis Excel Table

## Data Set
In the first Excel table "Jobs&Traits.xlsx" we have a bunch of data about each job using different criteria. 
In the second table "Tai info.xlsx" or for example "Dang info.xlsx" we have student information, grades under each criterion.
 
## Installation 
Install Poi Package (4.1.2)
In the src folder run StudentMatch.java
